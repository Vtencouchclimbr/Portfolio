## Setup Instructions
### Prerequisites
- Node.js (v14+)

### Installation
1. Clone the repository:
   ```bash
   git clone [Repo URL]
   ```
2. Navigate to the project directory:
   ```bash
   cd TodoList-React
   ```
3. Install dependencies:
   ```bash
   npm run install
   ```
5. Start the development environment:
   ```bash
   npm run dev
   ```
6. Access the app at `http://localhost:3001`.
